# Complete Hackathon II AI-Powered Todo Chatbot – Full-Stack Project

Welcome to the **AI-Powered Todo Chatbot**, an intelligent task management assistant designed to help users stay organized, productive, and on top of their daily tasks. This project is a complete full-stack application that combines personalized task management with an interactive AI assistant to guide users in creating, managing, and prioritizing their tasks.

---

## Project Overview

This application allows users to **sign up, log in, and manage their own tasks**. Every user has a private account, ensuring that tasks are personal and secure. The unique aspect of this project is the **AI Chatbot** that assists users in managing tasks more intuitively — whether it’s adding a new task, marking tasks as complete, or getting reminders about pending items.

The project also includes several **informative static pages** to enhance user experience and provide guidance.

---

## Core Features

- **User Accounts:** Users can create their own accounts and log in to access their personalized task lists.  
- **AI Chatbot Assistance:** Users can interact with a chatbot to add, update, or manage tasks using natural language commands.  
- **Task Management:** Easily add new tasks, update existing ones, mark them as complete, or delete tasks no longer needed.  
- **Task Organization:** Assign tasks to categories and set priority levels to help users focus on what matters most.  
- **Interactive UI:** The application presents tasks in a clean, organized interface with instant updates.  
- **User-Specific Data:** Each user sees only their own tasks, keeping personal data secure and organized.


1. **Sign Up:** Create a personal account to start managing tasks.  
2. **Log In:** Access your private dashboard where your tasks are stored.  
3. **Interact with the Chatbot:** Use natural language to add, update, or complete tasks effortlessly.  
4. **Manage Tasks:** Add tasks manually if preferred, categorize them, set priority levels, and mark tasks as complete when done.  
5. **Explore Static Pages:** Learn more about the app, its features, usage tips, and contact options.
